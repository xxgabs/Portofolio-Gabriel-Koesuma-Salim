Racing Wheel Customizer (Personal Project)

Racing Wheel Customizer is an interactive web application that allows users to customize wheel colors in real-time through a 3D visualization, enabling the wheel design to be previewed before production.

Technology

  • HTML
  • CSS
  • JavaScript
  • React
  • Next.js
  • Three.js / React Three Fiber

Features

  • 3D Wheel Viewer
  • Real-time Wheel Color Customization
  • Interactive Rotation and Zoom
  • Download Wheel Design Image
