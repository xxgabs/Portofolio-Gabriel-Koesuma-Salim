Mobile Application UI/UX Design Concept

This project is a UI/UX design concept for a mobile application created using Figma. The goal of this project was to design a clean, modern, and user-friendly interface that focuses on intuitive navigation and strong visual consistency.

Design Tools

  • Figma
  • Wireframing
  • User Flow Mapping
  • UI Prototyping

Design Process

The design began with planning the user journey and defining the main features of the application. Wireframes were created to outline the structure of each screen. After that, the final UI design was developed with a modern visual style to improve user experience and usability.
